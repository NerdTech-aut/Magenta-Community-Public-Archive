Hi @faceman

Den DNS Server alleine zu ersetzen geht nicht.

Du musst auch die DHCP Server Funktion an PiHole auslagern, denn der DHCP Server lässt sich an dieser Box deaktivieren. Und über DHCP bekommen deine Geräte mitgeteilt welchen DNS Server diese verwenden sollen.
