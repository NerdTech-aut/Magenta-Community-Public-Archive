Hey @ManKo ,
hier gibt's einige Unterschiede zur alten Box:

1. Carrier Aggregation = Sender mit unterschiedlichen Frequenzen können gebündelt werden und das kann dazu führen, wenn man die neue Box hat, dass das Signal verbessert wird
2. Die neue Box ist ein cat6 Gerät = Kann mehr Speed erreichen (bis zu 300 Mbit/s) - die alte hingegen ist ein cat4 Gerät
3. Die neue Box verfügt über 2,4Ghz und 5Ghz Frequenzband = wenn in der Umgebung viele auf 2,4Ghz surfen (wovon auszugehen ist) kannst du Verbesserungen mit 5Ghz erreichen
4. Die neue Box hat einen fest verbauten Akku
5. Die neue Box hat einen TS-9 Anschluss für die externen Antennen, während die alte SMA Anschluss hat

Es ist besser auf die neue Box umzusteigen, da diese bereits soweit aufgerüstet ist, damit sie mit der neuen Netztechnologie kompatibel ist.

LG Nadi
