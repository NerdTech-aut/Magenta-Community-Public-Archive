  • Geben Sie bei Name T-Mobile A Internet ein.
  • Geben Sie bei APN internet.t-mobile.at ein.
  • Geben Sie bei Proxy nichts ein ein.
  • Geben Sie bei Port nichts ein ein.
  • Geben Sie bei Nutzername t-mobile ein.
  • Geben Sie bei Passwort tm ein.

  • Scrollen Sie nach unten.
  • Geben Sie bei MCC 232 ein.
  • Geben Sie bei MNC 03 ein.
  • Wählen Sie bei Authentifizierungstyp PAP.
  • Wählen Sie APN-Typ.

Falls diese Seite angezeigt wird, geben Sie default,supl ein (Achtung: in Kleinbuchstaben) und wählen Sie OK.

Falls diese Seite nicht angezeigt wird, aktivieren Sie Internet und wählen Sie OK.
