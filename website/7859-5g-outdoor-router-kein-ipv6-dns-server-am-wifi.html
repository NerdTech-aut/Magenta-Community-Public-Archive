Hi,
der Outdoor Router mit Indoor WiFi AP ist ein geniales Produkt, um endlich ohne extra Aussenantennen guten Empfang in schnelles Internet zu verwandeln.

Aber leider hat der indoor Teil ein paar Probleme:
das Größte: 5G Netz kann wunderbar IPv6, der Outdoor Router unterstützt es und alle Geräte die direkt angebunden sind können es stabil nutzen.
Aber Devices die mit der Indoorunit via Wifi angebunden sind, bekommen keine DNS Server vom DHCP des AP zugewiesen.
via LAN des AP aber meist schon.
das führt via WiFi zu Problemen!
bitte um einen Firmwareupdate. die Aktuelle Version ist mit September 2021 datiert.

viele Grüße
