Hi @PedroKTFC

Have you already changed the APN?

Because the default APN only allows for your router to get a private IP address. But for port forwarding to work you need a public IP address.

With the other APN your router can get a public dynamic IP address. The dynamic means that it changes from time to time.

A step-by-step guide can be found here (in German):

https://www.magenta.at/hilfe-service/services/hardwaresupport/topic/sicherheit/apn-nat-typ-andern/device/internet-5g/box-mc801a

regards NTM
