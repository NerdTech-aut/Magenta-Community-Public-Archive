Hallo @Dnik_Five und willkommen in der Magenta Community,

Versuch folgendes:  Mit dem Code *#*#86583#*#* kann VoLTE aktiviert werden.

Mit dem Code *#*#869434#*#* kann VoWiFi aktiviert werden.

Vielleicht gibst du uns dann eine Rückmeldung, ob es mit der Eingabe der Codes nun funktioniert bzw. im Menü Punkt aus wählbar ist.
