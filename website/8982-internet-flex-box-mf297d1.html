Hallo @schlege ,

es tut mir leid, dass Sie Probleme mit der Verbindung über das LAN-Kabel zwischen Ihrer Flex Box MF297D1 und Ihrem WLAN-Router haben. Hier sind einige Schritte, die Sie ausprobieren können, um das Problem zu lösen:

1. Überprüfen Sie die Verkabelung: Stellen Sie sicher, dass das LAN-Kabel ordnungsgemäß in die LAN-Ports sowohl der Flex Box als auch des WLAN-Routers eingesteckt ist. Manchmal kann ein lockerer Stecker eine schlechte Verbindung verursachen.

2. LAN-Einstellungen überprüfen: Gehen Sie in die Einstellungen der Flex Box und überprüfen Sie, ob die LAN-Verbindung aktiviert ist und ordnungsgemäß konfiguriert wurde. Stellen Sie sicher, dass DHCP aktiviert ist, um automatisch eine IP-Adresse zu beziehen.

3. Router-Einstellungen überprüfen: Überprüfen Sie die Einstellungen Ihres WLAN-Routers, insbesondere ob die Ports, an die die Flex Box angeschlossen ist, aktiviert und funktionsfähig sind.

4. Kabelqualität überprüfen: Manchmal kann ein defektes oder minderwertiges Kabel zu Verbindungsproblemen führen. Versuchen Sie, ein anderes LAN-Kabel zu verwenden, um sicherzustellen, dass das Kabel selbst nicht das Problem ist.

5. Firmware-Update: Überprüfen Sie, ob für Ihre Flex Box ein Firmware-Update verfügbar ist, das möglicherweise bekannte Probleme behebt.

LAN-Treiber aktualisieren: Stellen Sie sicher, dass Ihre Netzwerkkartentreiber auf dem neuesten Stand sind. Gehen Sie auf die Website des Herstellers Ihrer Flex Box und suchen Sie nach aktualisierten Treibern.

6. Verbindungstest an einem anderen Ort: Wenn möglich, versuchen Sie die Flex Box vorübergehend näher an den WLAN-Router zu bringen und eine Verbindung über das LAN-Kabel herzustellen. Dadurch können Sie feststellen, ob das Problem tatsächlich mit der Entfernung zusammenhängt.

7. Technischer Support: Wenn alle oben genannten Schritte nicht funktionieren, empfehle ich Ihnen, den technischen Support unter 0676 200 7777 zu kontaktieren. Sie können Ihnen bei der Diagnose und Lösung des Problems helfen.

Ich hoffe, dass einer dieser Vorschläge Ihr Problem lösen kann.

LG RS.
