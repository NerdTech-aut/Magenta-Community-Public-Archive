Also du musst die IP Adresse der Fritzbox eingeben - diese hängt ja in irgendeiner Weise an der HomeNet Box

Die Fritzbox hat keine öffentlich IP, sondern sollte eine IP im Bereich 192.168.1. haben - Das findest du normal über das Admin Interface der HomeNet Box heraus. Dort siehst du alle verbundenen Geräte
