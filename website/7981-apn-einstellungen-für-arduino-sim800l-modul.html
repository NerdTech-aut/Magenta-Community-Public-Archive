Hey @MrNice
Mit einer Magenta Wertkarte würde der Magenta Standard-APN funktionieren (egal ob 2G/GSM, 3G/UMTS oder 4G/LTE):
const char apn[] = " internet.t-mobile.at "; // APN
const char gprsUser[] = " t-mobile "; // GPRS User
const char gprsPass[] = " tm "; // GPRS Password
