Hallo,
für alle Verträge von T-Mobile Privatkunden hat sich seit dem 26.02.2016 die Kündigungsfrist folgend geändert:

- meldet sich der Kunde ab dem 26.02.2016 einen neuen Vertrag an bzw. führt eine Vertragsverlängerung durch, so hat er eine Kündigungsfrist von einem Monat.
- hat der Kunde vor dem 26.02.2016 einen neuen Vertrag abgeschlossen bzw. eine Vertragsverlängerung durchgeführt, so hat er noch eine Kündigungsfrist von 3 Monaten.

LG Nadi
