Hallo Robert,

grundsätzlich wird der Zugriff über das DLan auf die WD My Cloud Home Festplatte funktionieren, aber Du musst Dir im Klaren sein, dass diese Platte kein NAS Server ist.

Die My Cloud Home ist eine Festplatte, die zwar bei Dir Zuhause steht, aber ausschließlich über den Cloud Service von Western Digital erreicht werden kann. Alle Daten fließen zunächst ins Internet zum Service von WD und kommen dann von dort auf Deine Festplatte. Auch wenn theoretisch von dieser Platte Videos gestreamt werden können, ist sie m.E. nur zur Ablage von Dateien, Fotos und Videos konzipiert um dann von allen mobilen Geräten darauf zugreifen zu können, was auch recht ordentlich funktioniert.

Sobald, keine Internetverbindung besteht, gibt es keine Möglichkeit auf die Platte zuzugreifen. Es hilft auch nicht, die Platte direkt an einen PC oder Mac anzuschließen, sie wird einfach nicht als Festplatte erkannt. Ein Zugriff von allen lokalen und mobilen Geräten geht nur, wenn auf diesen Geräten eine App von WD installiert ist und der User sich bei WD anmeldet. Spannend wird das Ganze, wenn WD mal beschließt seinen Cloud Service einzustellen, dann dürfen wohl alle Daten verloren sein.

Wenn ich Dich richtig verstehe, suchst Du eher einen Mediaserver, der auch in der Cloud funktioniert, und nicht eine reine Cloud Festplatte. In diesem Fall würde ich Dir ein „echtes“ NAS System empfehlen.
