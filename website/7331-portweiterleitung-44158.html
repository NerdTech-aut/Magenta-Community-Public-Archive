Hey @Basic90

Bei Internet Fiber gibt es standardmäßig eine öffentliche IPv6 Adresse.

Damit man mit diesem Anschluss allerdings Portweiterleitungen nutzen kann, muss der Anschluss mit IPv4 laufen.

Eine Umstellung auf IPv4 ist über den Kundenservice von Magenta möglich.
