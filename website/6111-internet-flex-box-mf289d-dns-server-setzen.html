Man kann den DNS Server schon manuell einstellen wenn man die Funktion über die Java Konsole einblendet ;-)

in den Einstellungen den DHCP Menüpunkt aufrufen, die Java-Konsole einblenden und dann $('#dhcp_dns').show(); eingeben.
