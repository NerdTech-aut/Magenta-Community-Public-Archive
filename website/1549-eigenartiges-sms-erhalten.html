Hallo!

Kurznachrichten vom SMS Absender 44044 sind spezielle SMS, die im Zuge eines Einkaufs/ Aboabschluss zum Thema digitale Güter von T-Mobile versendet werden. Der Empfang dieser SMS bzw. auch eine Antwort an diese SMS Nummer ist stets kostenfrei.

WICHTIG ist jedoch, dass der SMS Text genau gelesen wird, da es durchaus sein kann, dass unseriöse Anbieter von Klingeltönen, Dating Services, Erotikservices etc. hier einfach eine Bezahltransaktion initiieren und darauf hoffen der Kunde antwortet mit JA auf diese SMS.

Auch ist es möglich, dass diese SMS vom ABsender 44044 einen TAN Code (OneTimePin) enthält der auf einer Webseite eigegeben werden muss. Nur wenn es sich bei der Webadresse um die Adresse https://mein.t-mobile.at/payment..... handelt, ist man auf der sicheren Seite, da hier seitens T-Mobile die betreffenden Kauf/ Zahlungsbedingungen angegeben werden.

Logischerweise sollte man den TAN Code nur dann eingeben, wenn auch tatsächlich ein Contentservice angefordert wurde.

Wenn Du versehentlich mit JA auf die SMS geantwortet hast so wird bei diesen Content Diensten (digitale Güter) zumeist ein Abo angelegt. Die damit verbundene Abozustimmung kann jederzeit am Selfcare Portal von T-Mobile gelöscht werden. EInfach einloggen auf https.//mein.t-mobile.at

Danach auf den Menüeintrag "Rechnung" => Verwaltung digitale Güter.
