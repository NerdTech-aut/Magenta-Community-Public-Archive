Hey @gerwid
Läuft der Anschluss eventuell mit IPv6 statt mit IPv4?

Dies kannst du im Interface der FiberBox unter Admin -> Information -> WAN IP Information prüfen.

Damit die Option zum Umschalten in den Bridge Mode verfügbar ist muss der Anschluss mit IPv4 laufen.
Über den Kundenservice (technische Fragen) kannst du die Umstellung von IPv6 auf IPv4 beantragen.
